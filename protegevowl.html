ProtégéVOWL: VOWL Plugin for Protégé

VOWL Specification »

ProtégéVOWL is a Protégé plugin for the user-oriented visualization of ontologies. It implements the Visual Notation for OWL Ontologies (VOWL) by providing graphical depictions for elements of the Web Ontology Language (OWL) that are combined to a force-directed graph layout representing the ontology. ProtégéVOWL is based on VOWL 2.0, which focuses on the visualization of the ontology schema (i.e. the classes, properties and datatypes, sometimes called TBox), while it does not consider individuals and data values (the ABox).

Note that the current version of ProtégéVOWL does not implement all visual elements defined in the specification of VOWL 2.0. Further elements will be added to the next version of ProtégéVOWL.

One-sentence installation instructions: Download JAR file, copy it to plugins folder of Protégé, activate the plugin via Window → Tabs → VOWL.

Installation of ProtégéVOWL

  1. Install Protégé (if not already installed)
  2. Copy JAR file to the plugins folder of Protégé
  3. Start Protégé and activate the VOWL plugin
  4. Verify Installation
  5. Customize ProtégéVOWL

1. Install Protégé (if not already installed)

ProtégéVOWL has been developed for the ontology editor Protégé that needs to be downloaded and installed first. ProtégéVOWL requires at least version 4 of Protégé Desktop that is available at protege.stanford.edu free of charge.

2. Copy JAR file to the plugins folder of Protégé

ProtégéVOWL consists of a JAR file that can be downloaded here:

Download ProtégéVOWL plugin (beta 0.1) ↓

The JAR file needs to be copied to the plugin folder of Protégé, i.e. <installation_path>\Protege_4.x\plugins\.

3. Start Protégé and activate the VOWL plugin

After Protégé has been started, the VOWL plugin can be activated via the Window menu (see Figure 1).

4. Verify Installation

A new tab entitled VOWL is shown next to the other tabs in Protégé (see Figure 2).

If this tab is selected, the active ontology is visualized using the VOWL notation. If nothing is visualized, an ontology needs first to be opened with Protégé (e.g. by selecting Open from URI... in the File menu and entering the URI http://purl.org/muto/core#).

5. Customize ProtégéVOWL

The user interface of ProtégéVOWL is structured as follows (see Figure 2):

  • VOWL Viewer - displays the ontology visualization;
  • VOWL Sidebar - lists details about the selected element;
  • VOWL Controls - allow to adapt the force-directed graph layout.

The user interface can be costumized with the small icons in the upper right of the views. The views can also be individually (de-)activated via the Window menu (Window → Views → Ontology views → VOWL Controls/Sidebar/Viewer).
